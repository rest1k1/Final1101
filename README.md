# Final1101

## Описание проекта

Информационная система книжного магазина.

Проект включает:

- WPF-приложение
- ASP.NET MVC Web-приложение
- API

Функционал:

- просмотр товаров;
- авторизация пользователей;
- оформление заказов;
- просмотр списка заказов;
- работа с базой данных DBeaver Community.


## Используемые технологии

- C#
- WPF
- ASP.NET MVC
- Microsoft.EntityFrameworkCore 8.0.0
- DBeaver Community


## Запуск проекта

1. Открыть решение `Itogovaya_rabota_RPM.sln`.
2. Запустить нужный проект:
   - **BookStore**
   - **BookStore.Web**
   - **BookStore.Api**
3. База данных `Orders.db` находится в папке проекта и автоматически копируется в выходной каталог.
